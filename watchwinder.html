• Watch Winder

    Description

    A watch winder is a device that not only holds and displays an automatic watch, but also spins it, which turns the watches rotor, storing energy in the mainspring.



    MCU

    Powered by an Arduino Pro-Mini



    Features

    -Different operation modes selectable with momentary push button
    -Stepper motor
    -C++
